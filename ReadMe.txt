# Pousada Peninsula Buzios Website

Website for Pousada Peninsula Buzios, featuring:
- Home page with property overview
- About page with history and facilities
- Contact page with booking form
- Image gallery
- Responsive design

## Structure
- `index.html` - Home page
- `sobre.html` - About page
- `contacto.html` - Contact page
- `css/` - Stylesheet directory
- `js/` - JavaScript directory
- `images/` - Image assets directory
